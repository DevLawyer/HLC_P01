Cascading Style Sheets

¿What is Cascading Style Sheets?


CSS is a graphic design language for defining and creating the presentation of a structured document written in a markup language.
It is widely used to establish the visual design of web documents, and user interfaces written in HTML or XHTML; the language can be applied to any XML document, including XHTML, SVG, XUL, RSS, etc.

Ways to insert CSS code on your web page


  • First form:

    • The first way is in tags, or inline styles, this method can only be done within the tag in question, therefore it only affects that line.
    For example: < span style="color:#78DF97; background:#000000; ">Lunes < /span >
    Result: Lunes


  • Second form:

    • The second way is to put all the code in an internal style sheet, opening a tag < style >
    It is important to know that this tag goes inside the head tag.
    For example: < style > p{color:Green; padding:13px;} < /style >


  • Third option:

    • The third way is to add an external style sheet with the tag < link >
    With this form we can use the style sheet created in any document that you need. For example: < link rel="stylesheet" href="EjercicioHLC.css">
    It is important to know that this tag goes inside the head tag.

How to use colors in CSS.


  • Using "color" tag:

    • We can use a hexadecimal value to specify the color, or write the color name directly.
    Example using hexadecimal value: color:#78DF97;
    Example using the name of the color: color: Green;


  • Using "rgb()" tag:

    • In this case to specify the color, we must use a value in rgb.
    For example: color:rgb(255,0,0);


  • Using "rgba()" tag:

    • It has a difference with respect to the previous one, now we can specify a fourth value that will serve to modify the transparency.
    For example: color:rgb(255,0,0,0.5);
